##### 步骤4： 执行安装脚本

##### 解压
```
tar xzvf imooc-dmp-env.tar.gz
```
##### 进入到 imooc-dmp-env 文件夹

```
cd imooc-dmp-env
```
##### 环境初始化
**`会停顿20秒，不要手动终止`**
```
sh imooc-dmp-start.sh
```

##### hdfs 启动

```
sh hdfs.sh
```

##### yarn 启动

```
sh yarn.sh
```

##### hive 启动

```
sh hive.sh
```

##### hbase 启动

```
sh hbase.sh
```

##### spark 启动

```
sh spark.sh
```

##### nginx启动

```
sh nginx.sh
```
##### es启动
```
sh es.sh
```
##### clickhouse启动
```
sh clickhouse.sh
```
##### mysql启动
```
sh mysql.sh
```
##### redis启动
```
sh redis.sh
```
